# SchierTabManager
## 项目介绍
管理、导入、导出网站标签的浏览器插件，也可作为导航网站使用

SchierTab 聚焦于提升浏览器新标签页的使用效率与个性化程度，整合快捷访问、自定义布局、个性化样式等能力，同时保证扩展的轻量性与易用性，适配不同浏览器的扩展规范，满足用户对浏览器使用体验的定制化需求。

鼠标悬浮在标签上，支持关闭标签、添加标签到分组。  
快捷打开/关闭管理页：默认 Ctrl+Q，可能不生效，在浏览器扩展快捷键设置中自定义。  
支持拖动修改分组位置和标签位置。  
所有数据保存在本地，支持导入导出数据。  

Github地址：https://github.com/x2288809/SchierTabManager
